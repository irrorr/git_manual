# Установка Git

## _Linux:_

```linux console
yum install git
apt-get install git
zypper in git
```

## _MacOS:_

Установить по ссылке [Git для Mac OS X](https://git-scm.com/download/mac) или через консоль:

```macos console
brew install git
```
## _Windows:_

Установка по ссылке [Git для windows](https://git-for-windows.github.io/)
